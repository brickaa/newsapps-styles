Grid Specs

The grid layout files are located in layout/grid.

To include an element in a responsive grid design, add the class .container to the parent-most element. Do not put a .container inside another .container.

Content-Block

Content-blocks are used to ensure that an element does not bump against the edge of the screen on smaller devices or window-sizes. Add the class .content-block to add a 20px left/right margin at all times, and the class .content-block-full to add 20px left/.

The mixin content-block is also added to certain theme styles to ensure elements always have left/right margin on smaller screens.

Project Header, Lead Art

These styles can be found in theme/lead-banner.scss. To assign the URL for the lead art, go to helpers/variables. The default height of the banner is 225px, and it can also be changed in helpers/variables.
