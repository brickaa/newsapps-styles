Typography

Typography styles can be found in helpers/_typography.scss. The sizes correspond w/ HTML header elements, but those elements do not have any specific styles. For example, .xxl should be used on h1, but if you use an h1 without the class .xxl, it will not include the preset xxl font-size.

This is a class .font-xxl

This is a class .font-xl

This is a class .font-l

This is a class .font-m

This is a class .font-s
This is a class .font-xs
Example HTML
        
          <h1 class="font-xxl">This is a class .font-xxl</h1>
          <h2 class="font-xl">This is a class .font-xl</h2>
          <h3 class="font-l">This is a class .font-l</h3>
          <h4 class="font-m">This is a class .font-m</h4>
          <h5 class="font-s">This is a class .font-s</h5>
          <h6 class="font-xs">This is a class .font-xs</h6>
Example Sass
        
        // Typography
        // ==========
        $base-sans-serif: $helvetica;
        $base-serif: $georgia;

        // Font Sizes
        $base: 16;
        $xxl: 36;
        $xl: 28;
        $l: 24;
        $m: 20;
        $s: 14;
        $xs: 10;

        // Calculate pixels to rem units for font sizes
        @mixin font-size($f) {
          font-size: $f+px; // for IE8
          font-size: $f/$base+rem;
        }

        // Include this mixin to change the font-size at a breakpoint.
        // For example, you could set font-size: $pct to 78% on mobile.
        @mixin font-size-bp($bp, $pct) {
          @include media($bp) {
            font-size: $pct;
          }
        }

        body {
          @include font-size($base);
          font-family: $base-serif;
          line-height: 1.2;
        }

        h1,
        h2,
        h3,
        h4,
        h5,
        h6 {
          font-family: $base-sans-serif;
          font-weight: 800;
          margin-bottom: calc(10rem / 16);
        }

        // These classes can be applied to any element
        // to set the font-size.
        .font-xxl {
          @include font-size($xxl);
        }

        .font-xl {
          @include font-size($xl);
        }

        .font-l {
          @include font-size($l);
        }

        .font-m {
          @include font-size($m);
        }

        .font-s {
          @include font-size($s);
        }

        .font-xs {
          @include font-size($xs);
        }

Prose

The basic prose styles outlined below can be found in helpers/_typography.scss. The extension %prose can be applied to any element to add basic paragraph styles, and is already applied to .prose. The .prose also includes an extension of %content-block, which gives elements a margin to prevent them from bumping up against the edges of small screens. Full-screen media elements should not be contained within a .prose tag.

Headline, Byline

Use the <h1> tag in .prose for Headlines, and <p class="byline"> for bylines.

Cat Saves Planet from Apocolypse, Dies in Action

By Captain Planet | July 0, 2084

Example HTML
              
              <div class="prose">
                <h1 class="headline">Cat Saves Planet from Apocolypse, Dies in Action</h1>
                <p class="byline">By <a href="http://www.texastribune.org/about/staff/captain-planet/">Captain Planet</a> | July 0, 2084</p>
              </div>

Paragraph Headers

That ^ is an example of a .paragraph-header. Notice it will automatically transform the text to uppercase. You can also use these styles on other elements using @extend %paragraph-header.

Example HTML
              
              <h3 class="paragraph-header">Paragraph Headers</h3>

Captions

Captions for photos and other media elements are styled using prose p.caption. Images within .prose will fill 100% of the available width, unless otherwise specified.

This is a picture of a kitten.

Example HTML
              
              <img src="http://placekitten.com/800/400">
              <p class="caption">This is a picture of a kitten. </p>

Blocks

Add a div.block around elements that you want to highlight with a yellow border on top and bottom, such as block-quotes or embedded lists.

Block Quotes

Use the class .block-quote for the text of a block quote, and .block-quote-source for the attribution. Also, make sure to use HTML special characters for your quote marks! They won't be styled correctly if you forget. **Warning: The example code below is converting the special characters, so make sure to change it if you copy/paste.**

“You know what’s awesome? Cats. Cats are awesome.”

— Becca Aaronson, cat enthusiast

Example HTML
              
              <div class="block">
                <h2 class="block-quote">“You know what’s awesome? Cats. Cats are awesome.”</h2>
                <p class="block-quote-source">— Becca Aaronson, cat enthusiast</p>
              </div>

Explainer Text

Use the class explainer-text when you're explaining something, such as how to use the Hospitals Explorer or to define a glossary definition.

            
    <p class="explainer-text">Text goes here</p>
Div.Button

Here are examples of HR tags. The basic HR tag is 2px wide and gray. Add the class "thin" to make it 1px wide and the class "yellow" to make it classic Tribune yellow. It includes 2rem white-space.

Grid Specs

The grid layout files are located in base/scaffold.

To include an element in a responsive grid design, add the class .container to the parent-most element. You can also use "@extend %container" to add an outer-grid to any element that needs it.

Content-Block

Content-blocks are used to ensure that an element does not bump against the edge of the screen on smaller devices or window-sizes. Add the class .content-block to add a 20px left/right margin at all times, or add "@extend %content-block" to the parent-most element's styles.

Project Header, Lead Art

The banner styles are located in layout > headers > banner. The default height of the banner is 225px, and it can also be changed in helpers/variables.

Basic Banner

Project Title

Zingy subtitle to hook 'em in

Example HTML
      
      <div class="banner">
       <div class="grand-header-image"></div>
       <header class="grand-header">
        <h1><a href="/">Project Title</a></h1>
        <p>Short descriptive subtitle</p>
       </header>
      </div>
Example CSS/Sass
      
        /* Basic Banner */

        $banner-height: 600px;
        $grand-header-image-url: 'http://placekitten.com/1000/800';

        .banner {
          background-color: lighten($masthead-black, 15%);
          display: table;
          // height: 600px;
          // max-height: $banner-height;
          margin: 0;
          position: relative;
          width: 100%;

          .grand-header {
            @extend %content-block;
            background-color: transparentize($masthead-black, .6);
            bottom: 0%;
            display: block;
            padding: 1em;
            position: absolute;
            text-align: center;
            vertical-align: middle;
            width: 100%;

            a {
              color: $white;
            }

            //Project Title
            h1 {
              color: $white;
              font-family: $base-serif;
              font-size: 2em;
              font-weight: 800;
              line-height: 1.15;
              text-shadow: $black 3px 3px 0;
              margin-bottom: .25em;

              @include media($default-768) {
                font-size: 3.5em;
              }
            }

            //Project Subtitle
            p {
              color: $white;
              font-family: $base-sans-serif;
              font-size: 1em;
              letter-spacing: .0625em;
              line-height: 1.15;
              text-shadow: none;
              margin-bottom: 1.5em;;

              @include media($default-768) {
                font-size: 1.5em;
              }
            }

          }

          .grand-header-image {
            background-image: url($grand-header-image-url);
            background-position: center center;
            background-size: cover;
            bottom: 0;
            left: 0;
            position: absolute;
            right: 0;
            top: 0;

          }

        }

About, Contributors

Give yourself (and your teammates!) some credit. Styles for the About section are located in layout/components/about.

The .about class includes @extend %prose, %content-block and %paragraph-header. Even though the styles look similar to .prose, don't put your About section in a .prose tag. It also includes two grid mixins: features-layout($bp-about) to add whitespace on the sides of large screens, and list-grid ($bp-about, 12, 2) to allow two contributors per row on large screens.

Tables

Table styles are located in layout/components/tables. You must define which header is related to each data point, in order to have headers on Mobile. To do this, set the "data-title" inside the <td> tag, i.e. <td data-title="blank">.

Title One Two
Big Bird A B
Bert C D
Ernie E F 
    
<table>
  <thead>
    <tr>
      <th>Title</th>
      <th>One</th>
      <th>Two</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-title="Title">Big Bird</td>
      <td data-title="One">A</td>
      <td data-title="Two">B</td>
    </tr>
    <tr>
      <td data-title="Title">Bert</td>
